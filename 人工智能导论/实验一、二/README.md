# 实验一、二：专家系统与模糊推理

## 实验一：动物识别专家系统

- 通过知识库和特征推理，实现对动物（如金钱豹、虎、长颈鹿等）的自动识别。
- 主要内容包括知识库构建、特征提取、推理过程实现。
- 相关代码：`AI_232_202307020122/experiment1.py`

## 实验二：模糊推理系统

- 实现了模糊关系的 max-min 合成与模糊推理过程。
- 支持模糊矩阵运算、加权平均决策等功能。
- 相关代码：`AI_232_202307020122/experiment2.py`

## 运行方法

1. 安装 Python 3.x 环境。
2. 进入 `AI_232_202307020122/` 目录，运行对应实验的 Python 文件：
	```bash
	python experiment1.py
	python experiment2.py
	```
3. 可根据实验报告文档，补充输入输出说明和实验分析。
