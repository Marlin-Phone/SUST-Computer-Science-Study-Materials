# 操作系统期末复习说明

操作系统的期末考试无真题，大题考试的内容与[期末例题](期末例题.pdf)基本一致，建议将该套题目中的所有大题题型都学会！  

在bilibili中有很多相关的复习视频，可以搜索“操作系统 期末复习”进行学习。  
我在考试前复习的时候观看的up主合集：  
[乒乓时报 | 操作系统大题 纯干货](https://space.bilibili.com/3493123485337744/lists/2953342?type=season)  
[咩咩吖努努力 | 操作系统期末速成](https://space.bilibili.com/630467604/lists/4772481?type=series)  
两位up主的讲解都很不错，可以结合着看。  
学会这两份合集中讲解的题目，基本上期末考试的大题就完全没问题了。  

祝大家期末考试顺利通过！
