未提到的考题（20分）：
在NAS文件系统中分析：
挂载
挂载点
权限组

- 挂载：把远程 NAS 共享“嫁接”到本地目录，使客户端像访问本地磁盘一样读写。  
- 挂载点：本地那个空目录，挂上去后其内容即被 NAS 共享覆盖。  
- 权限组：白名单规则，决定哪些 IP/网段（可跨 VPC）能访问对应挂载点。  

背过即可得到100分。
