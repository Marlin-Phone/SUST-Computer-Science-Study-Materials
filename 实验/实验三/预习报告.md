# 实验三 预习报告 — VLAN 配置

## 一、实验目的
- 掌握交换机 MAC 地址的学习过程。
- 理解 VLAN 的基本工作原理。
- 了解并掌握 Access、Trunk、Hybrid 链路端口的基本配置与作用。

## 二、实验器材
- 交换机两台、多台主机/PC、网线。

## 三、相关理论与要点
- MAC 学习：交换机根据接收到帧的源 MAC 与入接口建立转发表项，用于后续转发决策（学习 + 老化）。
- VLAN：逻辑上将交换机端口划分到不同广播域，不同 VLAN 的主机默认不能互通（需三层设备或路由）。
- Access 端口：属于单一 VLAN，端口发送/接收未打标签（untagged）。用于连接终端设备。
- Trunk 端口：传输多个 VLAN 的流量，使用 IEEE 802.1Q 对帧打/解标签（tagged），用于交换机间或与支持多 VLAN 的设备相连。
- Hybrid：混合端口，兼具 tagged/untagged 的特性（厂商命令不同，课堂中理解其用途）。

## 四、常用命令
- 查看 MAC 表：`display mac-address` 或 `show mac address-table`。
- 查看 VLAN：`display vlan all` 或 `show vlan`。
- 查看端口简要状态：`display brief interface`。
- 查看接口详细信息：`display interface Ethernet1/0/24`。
