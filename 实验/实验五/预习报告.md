# 实验五 预习报告 — 配置 RIP

## 一、实验目的
- 加深对 RIP 协议原理的理解。
- 熟悉 RIPv1 与 RIPv2 的区别与配置方法。
- 掌握 RIP 路由的运行机制和维护命令（查看运行状态、静默接口等）。

## 二、实验器材
- 两台或多台路由器、若干台主机、网线、串口线。

## 三、基本概念与要点
- RIP 概述：RIP 是基于距离向量的路由协议，使用跳数作为度量，最大可达 15 跳，16 跳视为不可达。
- RIPv1：不携带子网掩码信息，仅适用于不使用变长子网掩码 (VLSM) 或主类网络。
- RIPv2：支持掩码信息、路由认证和组播更新（224.0.0.9），适用于现代网络。
- silent-interface（或 silence）：用于在指定接口上不发送 RIP 更新包（常用于连接主机的接口，避免不必要的广播）。

## 四、常用命令
- 启动 RIP 进程：
  - `rip` → 进入 RIP 配置视图。
- 指定网络：
  - `network 192.168.0.0`（按网段声明要参与 RIP 的直连网段）。
- 指定版本：
  - `version 2`（在 RIP 配置下切换到 RIPv2）。
- 取消汇总（在 RIPv2 中推荐）：
  - `undo summary`。
- 查看 RIP 状态：
  - `display rip`（查看 RIP 运行参数与已宣告网络）。
- 查看路由表：
  - `display ip routing-table`（检查 RIP 学习到的路由条目）。
- silent-interface 示例（避免在连接主机的接口上发送更新）：
  - `rip` → `silent-interface Ethernet0/0`
