## 实验一 Servlet技术及应用
该实验主要是学习Servlet的基本使用方法，包括处理GET和POST请求、文件下载以及配置Servlet。

**tomcat部署方法**：将 exp01 文件夹放入 tomcat 的 webapps 目录下，启动 tomcat 服务器后访问以下 URL。

### 实验一中可直接访问(get请求)的URL:

http://localhost:8080/exp01/input.html
http://localhost:8080/exp01/excel (下载excel文件)
http://localhost:8080/exp01/configDemo.do
http://localhost:8080/exp01/count
