## 项目介绍

陕西科技大学 计算机科学与技术专业 计算机网络课程的学习资料，已更新完毕，**仅供参考**。

本项目包含期末复习资料和实验报告等内容。[期末考试](期末考试)文件夹下详细说明了考试重点、考试真题和复习资料，正在复习的同学请详细阅读[README](期末考试/README.md)文件。  
实验文件夹下包含5次实验的预习报告和实验报告范例，供大家参考。
